Hi, welcome to my website. I'm Mike Spelling, a trainee developer for the BBC and this is my personal site. Take a look at some of the projects I've been working on in my free time. I'll update these whenever I get the time!

So far I'm working on 3 projects - 'Applet', 'Enigma' and 'Weather' as well as using this website to improve my ruby on rails knowledge and try some CSS (admittedly not great as yet!).

The first project is a Java applet in which I simply wanted to simulate gravity on 2D ball objects and allow mouse clicks to apply force. It works alright so far although collisions between balls is not perfect yet. Eventually I'll get this into a working game.

Another project I'm working on is recreating the Enigma machine in software. This was a task I started at the Software Craftsmanship 2012 conference with Gavin James and Aidy Lewis. I took the design further - to allow for any number of rotors and added a 'plugboard' which allows letters to be switched.

Finally there's the Weather app which gets your approximate location and uses an API to give you the weather in your area.

All the code for this ruby on rails app can be found on GitHub, so take a look if your interested, have a look around and feel free to leave a comment!
